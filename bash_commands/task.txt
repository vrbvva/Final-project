1. Перейти в домашнюю диркторию:
	cd ~
2. Создать новую папку
	mkdir new_folder
3. Перейти в созданную папку
	cd new_folder
4. Создать в папке новый файл
	touch QA.txt
5.  Перейти в текстовый редактор
	vim QA.txt
	#нажимаю кнопку "Insert"
	#пишу сочинение на тему "Как я провела своё лето"
6. Выйти из редактора, сохранив содержимое
	:wq
